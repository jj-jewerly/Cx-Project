
# 드론 커미셔닝 프로젝트

이 프로젝트는 Arduino와 Raspberry Pi를 활용하여 드론을 제어하고 데이터를 수집하는 시스템입니다.

## 구성 요소

- **Arduino Mega 2560**: 모터 제어 및 센서 데이터 수집
- **Raspberry Pi 4 Model B**: 데이터 처리, 카메라 제어, 통신 관리

## 파일 구조

- arduino_code/
  - drone_controller/
    - drone_controller.ino
- raspberry_pi_code/
  - main.py
  - drone_comm.py
  - camera_module.py
